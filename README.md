# Sistema Solar

Pequeno projeto HTML/CSS/JS que ilustra um modelo simplificado do Sistema Solar.

## Estrutura

- `index.html` - página principal com o Sol, planetas, órbitas, anéis e cinturão de asteroides.
- `solar.css` - estilos e animações.
- `solar.js` - comportamento interativo (nome do planeta ao passar o mouse/clicar).

## Visualização

### Local
1. Clone ou baixe o repositório.
2. Abra `index.html` em um navegador ou use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).

### Online
O site está publicado via GitHub Pages:

https://pedrindev019.github.io/sistema-solar/

(espere alguns minutos após o deploy inicial se não abrir de imediato)

## Como usar
- Passe o mouse sobre cada planeta para ver o nome no rótulo inferior.
- Clique em um planeta para ver um alerta com o nome.
